| Port | TX | RX | Usage |
|------|----|----|-------|

## Reserved Pins (DO NOT USE)

The following GPIOs are internally connected to the SX1262 LoRa transceiver:

| GPIO | Function | Conflict Note |
|------|----------|---------------|
| 3 | SX1262 MISO | — |
| 4 | SX1262 ANT_SW | **Was I2C SDA in v1.0!** |
| 5 | SX1262 SCK | **Was I2C SCL in v1.0!** |
| 6 | SX1262 MOSI | — |
| 7 | SX1262 NSS | — |
| 8 | SX1262 NRESET | — |
| 47 | SX1262 DIO1 | — |
| 48 | SX1262 BUSY | — |
